# PDF Note Space Adder

A Python tool to add extra note space on the right side of PDF pages with an optional grid pattern, similar to SpaceMyPDF.com.

## Features

- ✅ Add customizable note space on the right side of PDF pages
- ✅ Optional grid pattern
- ✅ Adjustable note space width (as percentage of page width)
- ✅ Works with multi-page PDFs
